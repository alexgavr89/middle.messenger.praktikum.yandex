# Яндекс практикум - модуль 1

- `npm start` — запуск версии для разработки, port 3000
- `npm build` — запуск сборки.
- `npm run serve` — запуск express, порт 3000.
- `npm run eslint` — запуск eslint.
- `npm run stylelint` — запуск stylelint.
